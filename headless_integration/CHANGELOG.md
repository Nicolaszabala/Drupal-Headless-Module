# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),

## [Unreleased]

### Added
- Initial module structure and configuration
- ConfigurationManager service for settings management
- ConsumerManager service for API consumer management
- Settings form with CORS and OAuth2 configuration
- Developer dashboard with system status
- Support for Drupal 10.3+ and Drupal 11
- Comprehensive test coverage (Unit, Kernel, Functional)
- Auto-configuration of JSON:API and OAuth2
- Rate limiting configuration
- Security best practices (CORS, OAuth2)
- Integration with Consumers and Simple OAuth modules

### Security
- CORS configuration with explicit origin whitelisting
- OAuth2 client credentials authentication
- Rate limiting capability
- Private file system requirement for OAuth2 keys
- Secure credential storage
